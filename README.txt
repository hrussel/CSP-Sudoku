ABOUT : 
	Developped for the Knowledge Representation course 2015 given by prof.dr Frank van Harmelen at the University of Amsterdam,
	this CSP solver solves problems with any domain
---------------------------------------------------
USAGE : my-csp-solver.py input_file [output_file]

	- input_file should contain one CSP problem per line.
	- ouput_file is optional. If missing, the solutions are printed on the standard output
---------------------------------------------------
CONSTRAINTS :
	- more "All different" constraints may be added to the problem in the function init_sudoku() from my-csp-solver.py
